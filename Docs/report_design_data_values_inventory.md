# Report Design Data Values Inventory

Purpose:
- provide a complete field inventory for designing the next report mock
- map customer, Supabase, derived, OpenAI, and unavailable values
- make it clear which labels can be shown confidently and which must be placeholders

Use this document as input when generating new report mocks.

## Report Personalisation Fields

These come from `public.recommendation_reports` and report generation inputs.

| Label idea | Field | Example | Notes |
| --- | --- | --- | --- |
| Prepared for | `customer_name` | `Sarah Smith` | Use in cover/header or intro copy |
| Customer email | `customer_email` | `sarah@example.com` | Usually avoid prominent display; useful for admin/audit |
| Report code | `report_code` | `SARAH-SMITH-20260504-001` | Good for footer or small metadata line |
| Report date | `report_date` / `created_at` | `May 2026` | Display month/year in header |
| Generated by | `generated_by_user_profile_id` | UUID | Internal; do not show in customer-facing mock |
| Source run | `recommendation_run_id` | UUID | Internal; do not show prominently |
| Report status | `report_status` | `draft` / `generated` | Internal/admin only |

Suggested personalization copy:

```text
Prepared for {customer_name}
Suburb investment report generated from your selected budget, out-of-pocket limit, and investment strategy.
```

| --- | --- | --- | --- |
| Estimated weekly OOP | `estimated_oop` | `((median_price * 0.8 * 0.06) / 52) - median_rent_weekly` | Current simplified estimate |
| Fits budget | derived from `median_price <= input_budget` | Yes/No | All returned suburbs pass this |

## OpenAI Context Fact Fields

Generated by Supabase Edge Function:

```text
supabase/functions/generate-suburb-report-summary
```

Cached in:

```text
public.suburb_ai_context_facts
```

Returned object:

```json
{
  "facts": {
    "nearest_major_city": {
      "name": "Bundaberg (regional city)",
      "distance_km": null,
      "evidence_url": "https://..."
    },
    "healthcare": [],
    "activity_centres": [],
    "local_drivers": [],
    "transport": [],
    "education": [],
    "infrastructure": []
  },
  "confidence": "medium",
  "data_limitations": []
}
```

Available context categories:

| Report label idea | OpenAI field | Example |
| --- | --- | --- |
| Nearby health infrastructure | `facts.healthcare[]` | `Bundaberg Hospital` |
| Activity centre / CBD | `facts.activity_centres[]` | `Bundaberg CBD` |
| Local drivers | `facts.local_drivers[]` | `Healthcare`, `Agriculture`, `Regional services` |
| Transport | `facts.transport[]` | `Regional airport`, `train station`, `highway` |
| Education | `facts.education[]` | `CQUniversity Bundaberg Campus` |
| Major infrastructure | `facts.infrastructure[]` | `New Bundaberg Hospital project` |
| Nearest major/regional city | `facts.nearest_major_city` | `Bundaberg regional city` |

Guardrails:
- Show citations subtly if space allows.
- Do not show uncited facts.
- Do not use OpenAI context facts as scoring inputs unless explicitly added later.

## OpenAI Report Summary Fields

Cached in:

```text
public.suburb_report_ai_summaries
```

Returned as `report_summary`:

```json
{
  "summary": "Max 2 short sentences.",
  "strengths": ["max 3"],
  "local_drivers": ["max 3"],
  "risk_notes": ["max 2"],
  "data_limitations": ["max 3"],
  "confidence": "medium"
}
```

Suggested placement:
- `summary`: near top of suburb page, after suburb name or near recommendation box
- `strengths`: key insight cards or positive bullets
- `local_drivers`: local context section
- `risk_notes`: risk/due diligence section
- `data_limitations`: small footnote or expandable note, not a dominant visual block
- `confidence`: small metadata chip; optional

| Median unit price | Not loaded | Show only if/when unit data is added; otherwise omit or show "not available" |

| Distance to CBD | Not loaded as structured field | Can be AI context only if cited, otherwise omit |
| Minutes by train | Not loaded | Do not show unless sourced |
| Price growth line chart | Historical series not loaded | Use placeholder or omit |
| Adelaide/metro benchmark line | Not loaded | Do not show unless benchmark source is added |
| Suburb image/photo | Not loaded | Use generated/placeholder image or sourced media with rights |
